HTML Proof

I've learn quite a lot about HTML through this activity I have learned that HTML or Hyper Text Markup Language is the standard markup language for creating Web pages and the structure of web pages. HTML consists of a series of elements and these elements tell the browser how to display the content. such as h1 to h6 is a header for web pages title is title for webpages ul is unordered list and many more to mention. The only challenge is you really need to understand all of the elements for you to be able to use is effectively. And my impression is that HTML is fun to do once you acquire enough knowledge about the language.

CSS Proof

CSS or Cascading Style Sheets, is a computer language for laying out and structuring web pages in HTML. CSS is mainly use to style and layout web pages for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features. The challenge in CSS is you have to familiarize the attributes so that you can get the right height, width, color, font and many more. My impression for CSS is that it is simple if you can familiarize the language.
